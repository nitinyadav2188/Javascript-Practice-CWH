JS INHERITANCE

  • is a mechanism that allows us to create new classes on the basis of already existing classes.
  • It provides flexibility to the child class to reuse the methods and variables of a parent class.
  • The javascript extends keyword is used to create a child on the basis of a parent class.

Points to remember:

  1. It maintains a IS-A relationship.
  2. The extends keyword is used in class expressions or class declarations.
