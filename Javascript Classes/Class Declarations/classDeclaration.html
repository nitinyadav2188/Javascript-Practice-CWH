Class Declarations:

  • A class can be defined by using a class Declaration.
  • A class keyword is used to declare a class with any particular name.
  • name of class always starts with uppercase letter.
Eg: class Employee

a class can be declared only once. Re-declaring it can throw an error.

Hoisting: unlike function Declaration,the class Declarationis not a part of javascript hoisting.So,it is required to declare the class before invoking it.
