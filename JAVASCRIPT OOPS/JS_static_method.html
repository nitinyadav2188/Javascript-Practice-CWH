JS Static Method:

Points to Remember:

  1. the static keyword is used to declare the static method.
  2. the static method can be of any name.
  3. a class can contain more than one static method.
  4. if we declare more than one static method with the same name,Javascript will always invokes the last one.
  5. static methods can be used to create the utility functions.
  6. we can use this keyword to call a static method within another static method.
